The Web Site consists of 3 pages (routes):
1. Home     (route: "/" or "/index.php")
2. About    (route: "/about.php")
3. Contacts (route: "/contacts.php")

Language:     PHP 7.4
Basic scheme: {host}:{port}/{domain_folder}/{route};   for example: http://127.0.0.1:8080/php-test-works/test-work-6/index.php

All errors are saving to a file logs/errors.log with err. codes;
Active page users (amount) are displayed on every page in real time.
